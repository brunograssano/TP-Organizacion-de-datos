# No se utiliza la de funcionesAuxiliares debido a que en este caso se tiene otro array. (en predict_proba [:,1]) (idem en mostrar el AUCScore)
def graficarROCCurve(modelo,nombreModelo,X_test, X_train, y_test, y_train):
    fpr_test, tpr_test, thresholds_test = roc_curve(y_test, modelo.predict_proba(X_test))
    fpr_train, tpr_train, thresholds_train = roc_curve(y_train, modelo.predict_proba(X_train))

    zero_test = np.argmin(np.abs(thresholds_test))
    zero_train = np.argmin(np.abs(thresholds_train))

    plt.plot(fpr_test, tpr_test, label="ROC Curve "+nombreModelo+" Test")
    plt.plot(fpr_train, tpr_train, label="ROC Curve  " + nombreModelo + " Train")
    plt.xlabel("FPR")
    plt.ylabel("TPR")
    plt.plot(fpr_test[zero_test], tpr_test[zero_test], 'o', markersize=10, label="threshold zero test",
             fillstyle="none", c="k", mew=2)
    plt.plot(fpr_train[zero_train], tpr_train[zero_train], 'x', markersize=10, label="threshold zero train",
             fillstyle="none", c="k", mew=2)
    plt.legend(loc=4)
    plt.show()

Redes Neuronales¶

Las redes neuronales estn dentro de lo que se considera modelos ms complejos. Este tipo de modelo dispone de una amplia cantdidad de parmetros que se pueden ir modificando hasta obtener los mejores resultados.

Para el preprocesamiento decidimos utilizar el mismo tipo de funcin que en otros modelos. Este preprocesamiento encodea las variables categricas mediante OneHotEncoding y normaliza las variables numricas.

In [11]:

In [12]:
X_train, X_test, y_train, y_test = train_test_split(X_redes_neuronales, y, test_size=0.25, random_state=0)

Los parmetros que tendremos en cuenta al entrenar las redes neuronales seran:

  • Funcin de prdida a optimizar: Es la funcin que se busca minimizar. En nuestro caso como deseamos hacer una clasificacin binaria, es decir, entre dos clases, utilizaremos siempre la funcin: binary cross entropy

  • Optimizador: Es el algoritmo mediante el cual se optimiza el funcin de perdida anterior. Utilizaremos Stochastic Gradient Descent (SGD) y Adam. La diferencia principal radica en que Adam es un optimizador que considera a la derivada segunda para realizar la optimizacin mientras que SGD solo la derivada primera. Esto puede llegar a suavizar la prdida al entrenar la red a lo largo de las pocas.

  • Tasa de aprendizaje: Es un parmetro que indica la velocidad con la cual el optimizador intenta acercarse el mnimo de la funcin de prdida. Una tasa pequea, requerir ms iteraciones para alcanzar el mnimo y una muy grande podra nunca encontrarlo, por ejemplo ya que se lo saltea constantemente.

  • Funcion de activacion de las neuronas: Es la funcin que se aplica sobre el input de cada neurona, antes de multiplicarla por el peso correspondiente. Hemos probado las siguientes: ReLu, sigmoidea y tanh

  • Cantidad de capas: Es la cantidad de capas de la red. Consideramos que teniendo una capa de input, una oculta y una de output era suficiente. Esto es porque agregando capas el tiempo de entrenamiento se volva poco razonable y posiblemente ms complicado de lo necesario.

  • Cantidad de neuronas de cada capa: Hemos entrenado con la siguiente configuracin de la red: La primera capa tiene 14 neuronas pues tenemos 14 features. La ltima capa tiene 1, lo cual nos servir para realizar la clasificacin en los 2.

Adems, en algunas redes, utilizamos dropout: durante el entrenamiento, algunas de las neuronas no se tienen en cuenta. Esto puede volver ms robusto al modelo, al hacer que la salida del mismo no depende nicamente de un camino.

En estas primeras redes vamos a ir probando sin aplicar regularizacion. Luego intentaremos ver si logramos mejorar el resultado agregandole las diferentes opciones.
